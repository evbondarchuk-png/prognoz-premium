# Прогноз — Премиум (v2)

Новая версия дашборда «Прогноз» для агентства «Этажи».

- **Прод:** https://premium.prognoz.info
- **v1 (не трогаем):** https://prognoz.info
- **Бэк:** общий с v1 — Firebase `prognoz-archive`, Cloud Functions Gen2, RTDB. Ничего не дублируем.
- **Стек:** статический HTML + ванильный JS, без сборки. Деплой — GitHub Pages, push в `main`.

## Дизайн

- Цвета: petrol `#0C5460`, gold `#A86A12`, зелёный/жёлтый/красный только для статуса.
- Шрифт: Golos Text (табличные цифры).
- Радиусы: 16 / 12.
- Референс: `partner-hybrid.html` из ТЗ-пакета.

## Правила

- Мозг — общий бэк, НЕ переписываем.
- Числа — из RTDB, не из макетов.
- Дефекты данных не чиним кодом — сообщаем Егору.
- Роли в интерфейсе: Партнёр / Старший партнёр / Управляющий партнёр / АУП.
