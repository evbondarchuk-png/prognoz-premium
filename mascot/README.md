# Маскот Прогноша — картинки эмоций

Сюда кладём рисованную лису по эмоциям. Файлы называть **строго так**
(assistant.js берёт их по имени; квадрат, лучше прозрачный фон, портрет «голова-плечи»):

| Файл | Эмоция | Когда показывается |
|---|---|---|
| `fox-proud.webp` | Гордая / восторг | ИР ≥ 90% или перевыполнение, серия хороших недель |
| `fox-happy.webp` | Довольная | ИР в норме (70–90%) |
| `fox-cheer.webp` | Подбадривает | ИР 45–70% |
| `fox-stern.webp` | Строгая, лапы скрещены | риск Н1–Н2 |
| `fox-alarm.webp` | Серьёзная тревога | Н3–Н4 / провал |

Пока файлов нет — показывается векторная лиса (фолбэк). Как появятся — подхватятся сами.

Формат: `.webp` (можно `.png`), ~512×512, один и тот же персонаж во всех эмоциях.
